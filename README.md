# PCL 夹克

## 安装
第一步：
```bash
pip install -r requirements.txt
```

第二步：

CUDA 10.2 & CUDNN 7.6 为最适配的版本

第三步：
```bash
python -m pip install paddlepaddle-gpu -i https://mirror.baidu.com/pypi/simple
```

## 准备
直播界面全屏化（1920x1080）

在CDK出现的地方应不存有任何遮挡物

## 使用


```bash
python main.py
```

识别、输入、领取CDK的过程是全自动的，请勿挪动鼠标
